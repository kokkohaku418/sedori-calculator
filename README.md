# Sedori Profit Calculator

無料のせどり利益計算ツール。Next.js (App Router) + TypeScript + Tailwind CSS。

## Development

```bash
npm install
npm run dev
# http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Deploy (Vercel)

1. GitHub にリポジトリを push
2. https://vercel.com/new からリポジトリを import
3. フレームワーク自動検出のまま Deploy

CLI の場合:

```bash
npm i -g vercel
vercel --prod
```
